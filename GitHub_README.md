# 화성시 사업담당자 계약업무 매뉴얼

화성시 빅데이터팀에서 개발한 **사업담당자용 인터랙티브 계약업무 매뉴얼**입니다.
공사·용역·물품 계약의 전 과정을 시각화하고, 18종의 계약 서식을 입력 정보 기반으로 자동 생성합니다.

🔗 **바로 사용** : [GitHub Pages 링크](#) (등록 후 자동 활성화)

---

## ✨ 주요 기능

### 🎯 계약 정보 입력 → 자동 산출
- **사업금액** 입력 + **네고율(%)** 조정 → **계약금액** 자동 계산
- 빠른 금액 버튼 (500만 / 1천만 / 2천만 / 3천만 / 5천만 / 1억)
- 계약 유형(공사 / 용역 / 물품)별 라벨 자동 변경
- 추가 조건 4종 (법인사업자 / 수의계약 / 관외업체 / 계약부서 협의)

### 📊 핵심 지표 자동 산출
- 계약 방식 판정 (수의 1인견적 / 수의 2인견적 / 일반경쟁입찰 / G2B)
- 계약 보증금 산출 (공사 7.5% / 용역·물품 5%)
- 검사자·입회자 결정 (3천만원 이상 시 팀장 + 회계팀장)
- 협조 결재 라인 자동 안내

### 📂 5단계 필요서류 자동 안내
1. **계약 전** — 견적서·사유서·내역서 등
2. **계약체결** — 청렴서약·면제각서·인감계 등
3. **착수/납품** — 착공계·납품내역서 등
4. **검사·준공** — 준공계·검사조서·정산합의서 등
5. **대가청구** — 대금청구서·완납증명 등

### 📄 18종 서식 자동 생성 + 다운로드
모든 서식에 입력 정보(계약명·금액·일자·업체·대표자·담당자·팀장·부서장)가 자동 반영됩니다.

- 견적서 / 수의계약 요청 사유서 / 수의계약 체결제한 확인서 / 수의계약 각서
- 관외업체 선정 사유서 / 계약보증금 면제 각서 / 청렴계약 이행서약서
- 입찰유의서 첨부생략 서약서 / 사용인감계 / 계약체결 필요서류 목록
- 착공계 / 공정예정표 / 준공계 / 준공검사원 / 정산합의서
- 납품내역서 / 물품 검사(수)조서 / 대금청구서

### 💾 데이터 영속성 (localStorage)
- **담당자·결재 정보 저장** : 한 번 입력하면 매번 다시 입력 불필요
- **사용자 주의사항 추가/수정/삭제** : 부서 특화 주의사항을 직접 관리

### 📥 다운로드 / 인쇄
- 개별 서식 HTML 다운로드 (입력 정보 자동 반영)
- 전체 서식 일괄 다운로드 (페이지 구분 포함)
- 브라우저 인쇄 → PDF 저장 가능

---

## 🚀 사용 방법

1. `index.html` 파일을 브라우저로 열거나 [GitHub Pages](#) 링크 접속
2. 좌측 패널에서 계약 정보 입력
3. 우측 필요서류 카드의 📥 버튼 클릭 → 미리보기 → 다운로드

### 💡 권장 환경
- Chrome / Edge / Whale 최신 버전
- 화면 해상도 1280×720 이상
